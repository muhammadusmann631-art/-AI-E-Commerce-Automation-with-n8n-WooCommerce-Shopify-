# 🛒 AI E-Commerce Automation with n8n (WooCommerce & Shopify)

## 📌 Overview

This project is an AI-powered automation workflow built using **n8n** for managing e-commerce operations across **WooCommerce** and **Shopify** stores. It automates order processing, order cancellation, and invoice generation with email delivery, reducing manual work and improving operational efficiency.

The workflow is provided as an **n8n JSON file**, making it easy to import, customize, and deploy.

---

## ✨ Features

* 🏪 WooCommerce store integration

## 🚀 Installation

1. Clone the repository:

```bash
git clone <repository-url>
```

2. Open your **n8n** instance.

3. Click **Import Workflow**.

4. Import the provided `workflow.json` file.

5. Configure your WooCommerce, Shopify, and Email credentials.

6. Activate the workflow and start automating your store operations.
